# Data Ingestion Module

This module is responsible for fetching, loading, and performing initial signal processing on the Predictive Maintenance dataset.

## Features
- **Dataset Download**: Automatically fetches the *AI4I 2020 Predictive Maintenance Dataset* from the UCI Machine Learning Repository if not present locally.
- **Data Loading & Cleaning**: 
    - Loads the raw CSV.
    - Standardizes column names (e.g., `Air temperature [K]` → `air_temp`).
- **Signal Processing**:
    - Calculates rolling window statistics (mean, standard deviation, variance) for telemetry signals:
        - Air Temperature
        - Process Temperature
        - Rotational Speed
        - Torque
        - Tool Wear
    - Handles sequence alignment by removing `NaN` values generated by rolling windows.

## Key Components

### `DataIngestor` Class
The main class to handle data lifecycle.
- `download_dataset()`: Downloads the dataset.
- `load_data()`: Loads and renames columns.
- `apply_signal_processing(df, window_size=5)`: Applies rolling window features.

## Usage
Run the script directly to generate `processed_telemetry.csv` in the `data/` directory:
```bash
python src/data_ingestion.py
```

## Output
- `data/ai4i2020.csv`: Raw dataset.
- `data/processed_telemetry.csv`: Cleaned data with rolling signal features.
