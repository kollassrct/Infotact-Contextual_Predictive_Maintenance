# Infotact-Contextual_Predictive_Maintenance
## Team Project

This project aims to build an advanced Predictive Maintenance system that combines internal IoT sensor telemetry with external environmental factors to predict machine failures before they occur.

Traditional predictive maintenance models rely only on machine-generated data. This project enhances prediction accuracy by integrating contextual information such as weather conditions, operational load, and environmental factors into the machine learning pipeline.

The final solution helps industries transition from reactive maintenance to proactive maintenance, reducing downtime and maintenance costs.
